## Installation

1. Clone the repository:
   ```
   https://github.com/muhammad-halimov/...
   ```
2. Navigate to the project directory:
   ```
   cd .....
   ```
3. Run the application:
   ```
   dotnet restore
   ```
   ```
   dotnet build
   ```
   ```
   dotnet run
   ```
